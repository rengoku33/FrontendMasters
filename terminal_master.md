# terminal
## movement
```text
ctrl + A     start of line  
ctrl + E     end of line  
alt + b      back one word  
alt + f      forward one word
```

## deletion
```text
ctrl + w     delete prev word  
alt + d      delete next word  
ctrl + U     detele from cursor to start  
ctrl + K     delete from cursor to end  
```

## search
```text
ctrl + r     search histoty with regex  
ctrl + t     fuzzy find all files and directrories (fzf)
```
